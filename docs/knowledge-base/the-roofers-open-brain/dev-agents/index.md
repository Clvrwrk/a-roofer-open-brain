# Dev Agents

OKF bundle for Open Engine and DevTeam agents, including runtime agents and support agents.

## Structure

- `concepts/` — durable concepts and policies.
- `sops/` or domain-specific folders — operating procedures when applicable.
- `decisions/` — accepted decisions and scope clarifications.
- `learnings/` — durable lessons from incidents, tests, and agent feedback.
- `validation/` — smoke tests and verification playbooks where applicable.
