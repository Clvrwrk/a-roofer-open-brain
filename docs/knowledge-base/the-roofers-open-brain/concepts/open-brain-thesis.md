# Thesis

The Roofers Open Brain is a property-first, roofing-native memory and agent operating system for Pro Exteriors.

The working model is:

- **The agents are the interface.** Humans interact with named agents in Slack and Command Center.
- **The brain is the asset.** Durable knowledge, evidence, SOPs, and decisions compound over time.
- **The data is the product.** Property-first, era-aware, evidence-backed data becomes the moat.

# Implications

- Knowledge belongs in OKF concepts, SOPs, decisions, and logs so future agents do not start from zero.
- Operational data belongs in Supabase and Command Center, with OKF concepts documenting semantics and access boundaries.
- Agents must stay within their plane and SOP boundaries; undefined work routes through Ops/Dev governance.
