# Open Brain Knowledge Base

This folder holds OKF-style markdown bundles for researched frameworks that should become reusable agent context.

- [Open Engine](open-engine/) - Linear-based operating surface for multi-agent work.
- [Open Skills](open-skills/) - Reusable, taste-bearing agent primitives and runbooks.
- [Application plan](application/framework-application-plan.md) - How to apply both frameworks to this repo, the Command Center, Supabase scale, security, and reliability.

Primary sources:

- [Open Engine](https://unlock-ai.natebjones.com/open-engine)
- [Open Skills](https://unlock-ai.natebjones.com/open-skills)
- [Open Knowledge Format blog](https://cloud.google.com/blog/products/data-analytics/how-the-open-knowledge-format-can-improve-data-sharing)
- [OKF v0.1 spec](https://raw.githubusercontent.com/GoogleCloudPlatform/knowledge-catalog/main/okf/SPEC.md)
