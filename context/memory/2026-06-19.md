  category sections verified on order 2011014809; no console errors.

  payload); order#/PO/job/branch search unchanged. Status change reloads (scope switch).

**Reusable pattern**: lazy-load line detail via API for any >~thousands-of-rows drill-down.
invoice-audit (1.23s, 2.6k lines) is acceptable; candidate for the same pattern if needed.
