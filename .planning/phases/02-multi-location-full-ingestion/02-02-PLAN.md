| Deno.env (secrets) → function memory | Secret values resolved at runtime; must never be logged or persisted |

| T-02-05 | Information Disclosure | Secret value logged or stored | mitigate | resolveKey resolves Deno.env.get(env_secret_name); only the secret NAME is ever referenced; skip-on-missing warns the NAME, never the value (hard rule 2); redactSample reused for any stored sample |
