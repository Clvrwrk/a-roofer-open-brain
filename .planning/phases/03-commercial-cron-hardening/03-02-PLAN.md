| Boundary | Description |
|----------|-------------|
| pg_cron/pg_net (DB) → acculynx-sync Edge Function | Async HTTP dispatch; the request_id is the only correlation handle across the boundary |
| net._http_response (transient, TTL 6h) → owned acculynx_cron_dispatch | Outcome data must cross before TTL purge or it is lost silently |
| trigger_acculynx_sync SQL fn → edge function | Carries the service-role Authorization header; must never be logged |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
